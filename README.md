# CBDC demo project with COSMOS blockchain

코스모스 블록체인을 이용한 CBDC 데모 프로젝트

# 파일 설명

CBDC_server :  코스모스 블록체인에 쿼리를 보낼 서버

CBDC_admin : 은행 관리 시나리오 데모

CBDC_user : 유저 서비스 데모

dpnmd : CBDC 코스모스 블록체인

# 실행 방법

shell 1 : 서버 실행
```
./run_CBDC.sh
```

shell 2 : Cosmos 블록체인 실행
```
starport serve
```
